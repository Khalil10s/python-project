# To-Do List in Python

A simple to-do list app that runs in the terminal.

## What it does

- Add, remove, and complete tasks
- Set priority (low, medium, high) and due dates
- Search and filter tasks
- Saves tasks automatically to `tasks.json`

## How to run

Make sure you have Python installed, then run:

```bash
python main.py
```

## Files

- `main.py` — the program
- `tasks.json` — saved tasks
